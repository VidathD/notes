# OSI Model

Data from applications is transformed into different formats until it is energy pulses.

## Physical Layer

About physical media. 
Cables, EM signals, fibre optics, NIC, Wi-Fi card.
Transmits data using forms of energy.

**NIC and Wi-Fi cards can have components belonging to data link layer.**


## Data Link Layer

Provides a unique physical addressing scheme for devices.
E.g. 48-bit MAC address. Usually written in hexadecimal format.


## Network Layer

IP addressing
