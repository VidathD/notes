## Print

- Basic print to STDOUT:
```python
print('{{String}}')
```

## Loops

### If-Else Loop

- Basic if-else loop:
```python
if {{condition}}:
    {{code}}
else:
    {{code}}
```

### For Loop

- Basic for loop:
```python
for {{iterator}} in {{something}}:
    {{code}}
```
- Using range:
```python
# i goes from 0 to {{integer}}-1
for i in range({{integer}}):
    {{code}}
```
### While Loop

- Basic while loop:
```python
while {{condition}}:
    code
```

## Arithmetic Operations

- Addition: 
```python
a+b
```
- Subtraction:
```python
a-b
```
- Multiplication:
```python
a*b
```
- Float division:
```python
a/b
```
- Integer division:
```python
a//b
```
- Division remainder:
```python
a%b
```
- Exponent:
```python
a**b
```
